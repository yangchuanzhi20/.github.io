2.assert

在Python中，assert 是一个关键字，用于在代码中加入断言。断言是一种声明，用于在代码中确保某个条件为真。如果断言的条件为假，那么程序将引发 AssertionError 异常。

请注意，当使用 -O（大写字母O）选项运行Python脚本时，断言将被禁用。这是为了在生产环境中避免执行断言检查。

  • graph_of_states 是模块或包的名称。
